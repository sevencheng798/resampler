# resampler
A Simple and Efficient Audio Resampler Implementation in C.
可以转换任意采样8000 ~ 48000

## 编译步骤
 
	在源码目录直接执行
	cmake && make
	将一个32Khz的单声道文件转换成16Khz单声道文件
	./resampler micin_1c_16bits_32Khz.wav micin_1c_16bits_16Khz_output.wav 16000
